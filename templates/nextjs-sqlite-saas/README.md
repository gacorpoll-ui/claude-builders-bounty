# CLAUDE.md Template: Next.js 15 + SQLite SaaS Project

This is an opinionated, production-ready `CLAUDE.md` context file for SaaS applications built with **Next.js 15 (App Router)** and **SQLite** (using Drizzle ORM and better-sqlite3 or Turso/libsql).

Place this file in the root directory of your greenfield Next.js + SQLite project so that Claude Code instantly understands your project's rules, patterns, commands, and architecture without requiring manual configuration or explanation.

---

## 📂 Files Included

- `CLAUDE.md` - The actual context template file containing project instructions, coding standards, database conventions, and dev commands.
- `README.md` - Setup and instructions on how to use and verify the template.

## 🚀 How to use in your project

1. Copy the `CLAUDE.md` file from this template folder to the root of your Next.js project.
2. If you are using **better-sqlite3** instead of **Turso/libsql**, or **Prisma** instead of **Drizzle ORM**, feel free to toggle the commented configurations in the `CLAUDE.md` file.
3. Open Claude Code in your project root, and it will automatically read the `CLAUDE.md` rules before executing any task!
